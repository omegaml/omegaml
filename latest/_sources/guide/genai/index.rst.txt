Generative AI
=============

omega-ml provides built-in capabilities for Generative AI workflows. This includes
retrieval augmented generation (RAG), document embeddings, text generation and agentic workflows.

Generative AI workflows require the following components, provided by omega-ml.

* *Generative AI Model* - A model to generate content, run RAG pipelines, use tools and process workflow steps
* *Model Provider* - Transparent access to a model provider, or hosting of your own AI models
* *Documents storage* - Document embeddings and vector DB
* *Monitoring* - tracking of model usage
