Classic ML
==========

omega-ml has been built from the ground up to support classic machine learning workflows.
This includes data preparation, model training, evaluation and deployment. Deployment scenarios
range from batch processing to real-time model serving via REST API or as part of an application.

To get started with classic ML workflows, follow these guides:

* :doc:`Loading datasets for training </guide/datasets/index>`
* :doc:`Training models </guide/models/index>`
* :doc:`Evaluating models </guide/models/metrics>`
* :doc:`Deploying models </guide/models/modelstore>`
* :doc:`Adding business logic to models </guide/models/businessmodels/index>`
* :doc:`Serving models for real-time predictions </guide/clusters/restapi>`
* :doc:`Tracking and monitoring models </guide/models/tracking/index>`
